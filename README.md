## 📊 考试概览

| 项目          | 说明                             |
| ------------- | -------------------------------- |
| **考试时长**  | 120 分钟                         |
| **题目数量**  | 100 道 (单选 70 题 + 多选 30 题) |
| **满分/及格** | 100 分 / 80 分                   |
| **题目分值**  | 每题 1 分                        |
| **考试形式**  | 线上/线下机考                    |
