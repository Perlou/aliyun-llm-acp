## 🔧 实践环境

### 开发工具

- **Python 3.10+**: 编程环境
- **VS Code**: IDE
- **Jupyter**: 交互式开发

### 云服务账号

1. 注册阿里云账号: https://www.aliyun.com/
2. 开通百炼服务: 控制台搜索"百炼"
3. 获取 API Key: 百炼控制台 → API-KEY 管理

### 本地开发

```bash
# 创建虚拟环境
python -m venv venv
source venv/bin/activate

# 安装依赖
pip install dashscope llama-index langchain openai

# 配置 API Key
export DASHSCOPE_API_KEY="your_api_key"
```
