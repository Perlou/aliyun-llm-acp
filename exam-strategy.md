# 🎯 阿里云大模型 ACP 考试攻略

## 📋 考试须知

### 基本信息

- **考试时长**: 120 分钟
- **题目总数**: 100 道
- **及格分数**: 80 分
- **题型分布**: 单选 70 题 + 多选 30 题

### 考试报名流程

1. 访问 [阿里云认证中心](https://edu.aliyun.com/certification/acp26)
2. 完成课程学习 (官方培训课程)
3. 进行模拟自测
4. 预约考试时间
5. 参加正式考试

## 🧠 应试技巧

### 时间分配策略

```
总时间: 120 分钟 = 7200 秒
每题平均: 72 秒/题

建议分配:
├── 单选题 (70题): 60 分钟 (~50 秒/题)
├── 多选题 (30题): 45 分钟 (~90 秒/题)
└── 检查修改: 15 分钟
```

### 答题顺序建议

1. **先易后难**: 快速完成有把握的题目
2. **标记跳过**: 不确定的题目先标记，稍后回来
3. **多选谨慎**: 多选题分值相同但难度更高，仔细审题
4. **不留空白**: 没有倒扣分，务必作答所有题目

### 多选题技巧

> ⚠️ **多选题是拉分关键！**

- 阿里云多选题通常选 2-4 个选项
- 排除法：先排除明显错误选项
- 关键词：注意"不正确"、"错误"等反向问法
- 完整性：检查是否遗漏正确选项

## 💡 常见陷阱题型

### 1. 概念混淆题

```
❌ 错误示例: RAG 和微调没有区别
✅ 正确理解: RAG 增强检索不改模型，微调改变模型参数
```

### 2. 参数范围题

```
❌ 错误示例: temperature 可以设置为 5
✅ 正确理解: temperature 范围通常 0-2
```

### 3. "以下哪些正确" 多选题

```
⚠️ 注意: 可能有 2-4 个正确选项
💡 策略: 每个选项独立判断，不要互相影响
```

### 4. 反向提问题

```
⚠️ 关键词: "不正确"、"错误"、"不属于"
💡 策略: 圈出关键词，选择错误选项
```
