### 考前准备

- [ ] 准考证（如有）
- [ ] 提前 30 分钟到达考场
- [ ] 手机静音或关机

### 考试技巧

1. **浏览全卷**: 开考后先快速浏览，了解题目分布
2. **先易后难**: 不确定的题目先标记，最后再回来
3. **时间分配**:
   - 单选 (70题): 约 60 分钟
   - 多选 (30题): 约 50 分钟
   - 检查: 10 分钟
4. **多选策略**: 不确定时选 2-3 个选项更安全
5. **保持冷静**: 遇到难题不要慌，先跳过

### 重点公式速记

```
RAG = 解析 → 切片 → 向量化 → 检索 → 生成
ReAct = Thought → Action → Observation
四层防护 = 输入 → 模型 → 输出 → 运营
RAGAS = Faithfulness + Relevancy + Precision + Recall
LoRA 参数 ≈ 0.1% 全参数
```
